Remote Claude Code

I love Claude Code. I also need to use the bathroom sometimes. So I built a remote app that lets me keep talking to Claude from my phone. Your Mac or Linux server runs Claude Code, your phone connects over WebSocket. Browse files, see git changes, view images, use voice, run commands. Everything you can do from your laptop, from your pocket. Uses your existing Claude Code subscription.
