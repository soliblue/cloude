# Widget UI Polish

Iterate on the visual design and UX of chat widgets. The core architecture and all 14 widgets are working — this is about making each one feel polished and native.

## Areas to Improve
- Error Correction: error words still slightly distinguishable from normal text (button vs plain text rendering)
- Consistent spacing and padding across all widgets
- Animations — add subtle transitions for state changes (check, reveal, restart)
- Dark mode testing and tuning
- Accessibility — VoiceOver support for interactive widgets
- Widget header buttons could become a shared reusable component instead of copy-pasted per widget

## Widgets (14 total)
Function Plot, Fill in the Blank, Interactive Function, Flashcard Deck, Quiz, Ordering, Matching, Categorization, Word Scramble, Sentence Builder, Highlight Select, Error Correction, Type Answer, Step Reveal
