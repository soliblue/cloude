# Android Voice Input & Transcription {mic}
<!-- priority: 10 -->
<!-- tags: android, audio -->

> Record audio, send to agent for Whisper transcription, insert into chat.

## Desired Outcome
Tap-to-record button in input bar, audio waveform visualization during recording, send base64 audio via `transcribe` message, receive `transcription` response, insert text into input field.

**Files (iOS reference):** AudioRecorder.swift, GlobalInputBar+Recording.swift, GlobalInputBar+AudioWaveform.swift
