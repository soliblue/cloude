# External Integration Skills {bolt.horizontal.circle}
<!-- priority: 8 -->
<!-- tags: skills -->

> Add a small set of high-value external skills such as browser automation, Spotify, and WhatsApp.

## Problem

Cloude is strongest inside the app, but still weak at acting through external consumer tools and communication channels.

## Candidate Skills

### 1. Browser automation
- navigate pages
- extract content
- screenshot pages
- eventually click and fill forms
- likely start simple, then add deeper browser control later

### 2. Spotify
- playback control
- now playing
- search and queue management
- combine AppleScript for local control with API support only if needed

### 3. WhatsApp
- read recent messages
- send messages with confirmation
- summarize important chats
- strongest communication upside, but also highest complexity and risk

## Rules

- Start with the smallest viable integration.
- Prefer local or user-controlled auth flows.
- Require explicit permission before sending or posting anywhere.
- Evaluate each skill on real usefulness, not novelty.

## Desired Outcome

A small set of external skills that materially expand what Cloude can do outside its own UI.
