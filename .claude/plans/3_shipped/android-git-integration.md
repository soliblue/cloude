# Android Git Integration


## Desired Outcome
View git status (branch, staged/unstaged files), view file diffs with syntax-colored additions/deletions, commit selected files with message. Uses `gitStatus`, `gitDiff`, `gitCommit` messages.

**Files (iOS reference):** GitChangesView.swift, GitDiffView.swift (+components)
