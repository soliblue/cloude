# Context Menu Tap Fix {hand.tap}
Replaced UIKit `WindowDismissOverlay` with SwiftUI dismiss layer so context menu buttons (Copy, Select, Collapse, Play) respond to taps instead of being intercepted by the dismiss overlay.
