# Majorelle Theme

New dark theme for the iOS app inspired by Jardin Majorelle and YSL.

## Palette
- Deep indigo-black base (`#0C0F1F`) - more purple than Ocean Dark's navy
- Rich ultramarine surfaces (`#1E2750`) - evoking the cobalt blue garden walls
- Icon: `leaf.fill` (garden reference)
- The existing coral accent color complements as terracotta

## Changes
- `Theme.swift`: Added `.majorelle` case with palette, icon, color scheme
- Automatically appears in the theme picker grid
