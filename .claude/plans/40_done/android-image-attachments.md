# Android Image Attachments {photo}
<!-- priority: 10 -->
<!-- tags: android, input -->

> Attach images to chat messages (camera, gallery, clipboard).

## Desired Outcome
Image picker from gallery/camera. Paste from clipboard. Image thumbnails in input bar. Send as base64 in chat message. Display received images in message bubbles.

**Files (iOS reference):** GlobalInputBar+ImageAttachments.swift, ImageEncoder.swift, MessageBubble+Components.swift
