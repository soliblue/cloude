# File Preview Smaller Text

Reduced font size in all text-based file previews from 12-13pt to 10pt for denser code viewing on mobile.

- Source code view: 12pt to 10pt (line numbers + highlighted/plain text)
- JSON tree view: 13pt to 10pt
- CSV table view: 13pt to 10pt (headers + data rows)
- Line number row height adjusted to 13.5pt to match new font size
- Added explicit lineSpacing(1.5) on text content to keep line numbers aligned with code lines
