# Memory

Write one concise document per serious round.

Rules:

- Include failed and rejected rounds, not only accepted ones.
- Keep each file short and compressive.
- Record traps, false assumptions, decisive signals, and safe patterns.
- Read the full folder before any non-trivial unguided round.
- Use `templates/round-memory.md` as the default shape.

Use this folder for reusable lessons.
Use `.claude/plans/` for the detailed round record.
