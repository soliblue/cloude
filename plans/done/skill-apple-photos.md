# Apple Photos Skill {photo.on.rectangle}
<!-- priority: 5 -->
<!-- tags: skill, integration, apple -->

> Access Photos.app via AppleScript. List albums, browse recent photos, export by date/album. iPhone photos sync via iCloud.

## Permissions Needed
- Automation → Photos.app (prompted on first use)

## Limitations
- Cannot search by content/faces — only by date, album, name
- Slow on large libraries (~40s)
- Read-only — cannot delete or edit

## Status
- Scripts built, not yet tested (permissions needed)
